Hermass 不是“给你一堆股票”的系统，而是把每天的赚钱动作压缩成三个问题：今天该从市场切，还是从个股切；这只票是继续、等待还是回避；什么时候回来复看，什么条件下直接失效。

市场判断是背景音，不是开关
先看路径，不先看指标
裁决比材料更重要
观察账本负责驱动复访
规则 1：先决定从哪进 顺风日先看市场与方向，逆风日先看局部强结构和外部线索。
规则 2：研究页必须给结论 不是读材料，而是先看到“继续 / 等条件 / 暂不继续”。
规则 3：观察对象要有时间感 什么时候回来，何时失效，何时触发，全部前置。
规则 4：复盘才会提升胜率 每一次裁决都要能回看，不然系统永远停留在表面聪明。

页面分工：每个页面只做一件事

不要在一个页面完成所有工作。Hermass 的页面各司其职，顺序用对了才是赚钱工具。

各司其职
/ 首页

只解决一件事：今天从哪条路进。看"顺风还是逆风""先看市场还是先看局部"，不要在首页完成研究。

/market 市场

适合顺风日、共振强、方向清晰时先看。用途是缩圈，不是直接下单。

/recommend 机会

适合逆风日、震荡日、局部机会集中时先看。从局部强结构里挑样本。

/research 研究

只干一件事：给个股裁决。先看"继续/等条件/暂不继续"，再看为什么、等待条件、失效条件、入场条件。

/watchlist 观察

不是收藏夹。只放真正需要复看的对象，每个必须有等待条件、失效条件、回来看的时间点。

/playbook 秘籍

相当于总说明书，用来校准使用顺序和避坑。

市场 vs 个股：双路径赚钱框架

先判断今天更适合自上而下还是自下而上。赚钱不是死盯一只票，而是在对的环境里用对的进入方式。

路径选错，后面全错

路径一：先看市场，再选个股

适合顺风或共振较强的日子。先缩圈，再深挖，不要一开始就埋进个股细节。

  1. 市场环境判断 看宽度、活跃性、风险偏好和资金面，确认今天是进攻日还是防守日。
  2. 方向识别 只盯少数承接最好的行业，不在弱方向里找侥幸样本。
  3. 机会池收敛 从推荐页挑局部最强对象，再进入研究页做裁决。
  4. 输出 市场观点 + 方向配置 + 少数重点对象。
互相验证

路径二：聚焦个股，反推验证

适合震荡、偏弱或外部线索驱动的日子。先研究样本，再回头校验它是不是逆风里的假机会。

  1. 样本本体研究 看结构、估值、增长、题材、资金与位置。
  2. 给出裁决 明确继续、等待还是回避，别把材料堆给用户自己拼。
  3. 补环境校验 用市场页和行业页确认它是不是孤立脉冲。
  4. 输出 买卖区间、风险边界、何时回来看。

研究裁决卡：赚钱不是看资料，是看结论

研究页第一屏必须先告诉你这只票应该怎么处理，然后再展开证据。三种结论足够覆盖绝大多数动作。

先结论，后证据
继续跟踪

能继续

  • 多周期结构至少两层站得住。
  • 资金确认或行业承接至少有一层成立。
  • 研究页要给出下一步动作：继续下钻证据，还是创建观察任务。
等条件

先别急

  • 现在更像观察样本，不是成熟机会。
  • 页面必须写清等待什么：资金确认、行业承接，还是 1-3 天站稳。
  • 等条件比强行下判断更赚钱，因为它直接过滤掉低质量试错。
暂不继续

先回避

  • 出现资金背离、跌回关键支撑、或结构明确失效。
  • 研究页要写清楚失效条件，不许只说“风险较大”。
  • 回避不是错过机会，而是保护注意力和本金。

研究页怎么用：按顺序看，不要跳

研究页第一屏必须按这个顺序看，不能先翻证据再回来看结论。

阅读顺序
1
结论

继续跟踪 / 等条件 / 暂不继续，这是第一眼必须看到的结果。

2
为什么

结构证据、资金证据、行业证据，理解为什么是这个结论。

3
入场条件

具体什么情况下可以继续执行，不要模糊的"等回调"。

4
风险边界

跌到哪里算失效、资金背离到什么程度要放弃。

5
等待/失效

等什么条件才升级？什么条件触发立即降级或删除？

继续跟踪 → 怎么做

结构和证据已经够了。继续下钻，必要时创建观察任务。不要看完结论就关页面。

等条件 → 重点看什么

现在不是坏，只是不成熟。重点看"等什么"：资金确认、站稳、行业承接。不要因为等条件就放弃好对象。

暂不继续 → 果断放弃

已经出现反证。不要恋战，不要因为看了很多材料就舍不得放弃。注意力比错过一只票更贵。

什么时候创建观察：四个条件缺一不可

不要先建观察再回头想为什么。只有满足这四条，观察任务才有意义。

门槛
1. 已有裁决

研究页已经明确"继续跟踪"或"等条件"。不给结论不建观察。

2. 知道看什么

你知道下次回来要看什么，不是"再看看有没有机会"。

3. 有失效线

知道什么情况下这条观察应当失效，不是无限期挂着。

4. 真会回来

你愿意在未来几天真的回来复看。不满足这四条，就不要建——那不是观察，是信息囤积。

赚钱六步法

把页面使用顺序固定下来，系统才能变成赚钱工具，而不是信息游乐场。

Playbook
1
界定问题

今天先问路径，不先问“买哪只”。先识别环境，再决定入口。

2
收集信号

市场页看环境，推荐页看收敛后的对象池，不平均用力。

3
分析研究

研究页给结论、等待条件、失效条件，而不是只堆资料。

4
形成观点

把对象定为“继续 / 等条件 / 暂不继续”，才算完成一次判断。

5
执行跟踪

创建观察任务，把提醒、有效期、复看时点写清楚。

6
复盘优化

回看哪些判断赚钱，哪些判断浪费注意力，持续调参。

观察账本：赚钱靠复看，不靠记忆

真正的观察对象必须带着时间感存在。不是“收藏”，而是告诉你什么时候回来、看什么、什么情况下直接删除。

Watchlist
回来看的时点

今天刚触发、这两天要复看、还是几天内到期，必须一眼看到。

等待条件

不是“继续关注”，而是写清楚等待资金确认、行业承接还是结构继续展开。

失效条件

跌回支撑、资金背离扩大、或突破失败，任何一条触发都要降级或删除。

下一次提醒理由

系统要告诉你为什么要回来，而不是让你自己想“这票我是不是看过”。

赚钱核心

Hermass 最值钱的不是数据多，而是让你少做三件亏钱的事。

少在逆风里强行找全面进攻机会 逆风日也会有机会，但那是局部、精选、证据化的机会，不是普涨幻想。
少把阅读误当成决策 看完一屏屏材料，不等于知道下一步该买、该等还是该撤。
少让机会失联 真正赚钱的对象通常不是当天看完当天结束，而是后面几天被持续正确跟踪。

观察账本怎么用：像处理待办，不像翻收藏夹

每天回来先看 /watchlist，不是先回 /recommend。按正确顺序阅读，不要让观察对象积灰。

阅读顺序
1
今天先看

哪些对象标记为"今天需要复看"，优先处理这组。

2
回看时点

哪些这两天该回来复看了，不要错过窗口。

3
等待条件

需要等什么信号才能往下一步走。

4
失效条件

什么情况直接删除，什么情况降级为"暂不继续"。

5
最近变化

上一次复看以来有什么关键变化。

6
下次理由

系统告诉你为什么下次要回来，而不是让你自己猜。

收盘后复盘三件事：看哪些对象从"等条件"变成"继续"；看哪些触发失效；删除不再成立的观察；只保留还值得复看的对象。

每天的推荐使用节奏

不同时段做不同的事，不要把开盘前的节奏用在盘中，也不要把盘中的焦虑带到收盘后。

节奏
开盘前或盘后首次打开

1. 首页看路径 → 2. 市场页或机会页缩圈 → 3. 研究页给 1-3 个对象下裁决 → 4. 需要复看的对象进入观察账本。

盘中或当天二次回来

1. 先看观察账本 → 2. 只处理"今天先看"或"刚触发"的对象 → 3. 不重新从首页开始翻全站。

收盘后复盘

1. 看哪些对象从"等条件"变成"继续" → 2. 看哪些对象触发失效 → 3. 删除不再成立的观察 → 4. 只保留还值得复看的对象。

观象怎么问，才有用

问题越接近"路径、裁决、追踪"，系统越有用。不要问它做不到的事。

提问技巧
不要问：今天买什么？ Hermass 不荐股，不执行交易。
要问：今天我该先看市场还是先看个股？ 路径问题，观象最擅长。
要问：这只票为什么是等条件，不是继续跟踪？ 裁决问题，让观象解释判断逻辑。
要问：这条观察的失效条件是什么？ 追踪问题，把模糊的"再看看"变成具体条件。
要问：我现在观察账本里先处理哪几个？ 优先级问题，让观象帮你排复看顺序。

最容易亏钱的六个误用

如果用户这样用系统，再漂亮的页面也赚不到钱。这一页要反过来教用户怎么避坑。

Do / Don’t
误用表现正确用法
把市场当开关一看大盘弱，就以为今天完全不能研究个股。市场只是背景音，决定你先从市场切还是先从线索切。
把推荐池当购物车看到一堆候选就想全看，注意力被打散。先看哪一组，再挑其中少数对象，不平铺几十只。
把研究页当资料库阅读很多，但最后没有明确动作。先看裁决卡，再决定要不要展开证据层。
把观察账本当收藏夹对象越来越多，但没有复看节奏。每个观察对象都必须有等待条件、失效条件和回来时点。
先问路径再问个股知道今天该从哪进，后面所有页面都会更有效。首页先看路由，顺风看市场，逆风看局部，迷茫时问观象。
先裁决再执行继续、等待、回避三分法会显著降低冲动试错。不给结论不下单，不写观察条件不创建观察任务。
一句话工作流

首页定路径，研究页下裁决，观察账本管复看。不要反过来，不要跳步骤。

核心原则

Hermass 不是"选股列表"，而是一个决策流程系统。先走对路，再下裁决，最后追踪复看。

最重要纪律

不要一上来就翻个股资料，不要看到候选池就想全看，不要把观察账本当收藏夹。

每日一句话

先走对路，再盯对票，最后用纪律把正确判断变成复利。
