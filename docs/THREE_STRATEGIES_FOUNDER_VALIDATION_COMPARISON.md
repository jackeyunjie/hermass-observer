# 三策略创始人验证对比

版本：v1.0 | 日期：2026-05-24

> 归档声明：本文档保留为历史研究材料，用于记录创始人/思想源头交易与 Hermass 状态框架的类比分析。它不属于当前 A 股生产系统的活跃设计依据、可信度主证据或日常运行入口。

---

## 系统定位

Hermass Observer 用三个经典趋势策略（VCP / 2560 / 布林强盗）分别对应趋势生命周期的三个阶段。为了验证系统的 State 环境匹配逻辑是否与策略创始人的实际交易一致，我们收集了三位创始人/思想源头的公开真实交易记录，用 Hermass State 框架进行回查分析。

## 核心发现

### 发现 1：三位创始人的交易都集中在"最佳适配"环境

三位创始人/思想源头的真实交易中，**72%-79%** 落入 Hermass State 系统定义的"至少双周期 E/F 共振 + 收缩后释放"环境。这不是巧合——它验证了 State 系统的环境识别逻辑与经典策略理论的底层一致性。

### 发现 2：止损逻辑高度一致

三位创始人都使用"技术位止损"（跌破支撑即出），而非固定百分比止损。Hermass 系统的止损规则（VCP: 收缩低点下方 / 2560: MA25 跌破 / 布林强盗: 递减均线）与创始人逻辑一致，且更为系统化。

### 发现 3：出场规则是共同的优化空间

三位创始人的出场策略都比当前系统更灵活（Minervini 的快速获利了结、Darvas 的箱体跟踪止损、Bollinger 的 Squeeze 级别管理）。这是系统后续迭代的主要优化方向。

| 验证维度 | 结论 | 可信度 |
|----------|------|--------|

| 策略-环境匹配 | 三策略的最优环境假设得到创始人交易数据的外部支持 | 高 |
| 止损规则 | 创始人止损逻辑与系统规则一致，系统更系统化 | 高 |
| 出场规则 | 系统有优化空间，但基础逻辑与创始人一致 | 中 |
| A 股适用性 | A 股本土化案例的双周期 E/F 占比（87-100%）高于历史美股类比数据 | 中高 |

## 数据来源与方法说明

- **历史美股交易**：采用**类比映射**方法（Hermass State 数据覆盖 A 股，历史美股交易仅作概念框架类比，不纳入当前系统证据链）
- **A 股交易**：采用**直接回查**方法（读取 Foundation DB 中的实际 State 数据）
- **可靠性**：创始人交易记录来自公开出版物和权威复盘，A 股案例来自公开社区分享（未经独立审计）
