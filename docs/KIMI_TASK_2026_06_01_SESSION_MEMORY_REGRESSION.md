# Hermass 会话记忆回归测试 — 第二轮行业追问验收

部署完成后，在浏览器控制台执行以下 2 步验证：

---

## 第1步：首轮问诊

- 发送：`"000021 怎么样"`
- 断言：返回里有 `remembered_stock_code=000021.SZ`

---

## 第2步：第二轮追问行业

- 同一会话（不要刷新页面），发送：`"它是什么行业"`
- 断言1：`session_id` 与第1步相同
- 断言2：`remembered_stock_code=000021.SZ`
- 断言3：`answer` 文本里包含"电子"（000021 所属行业）

---

## 失败判定

如果第2步 `answer` 里没有"电子"，而是返回全市场行业列表（如"当前行业覆盖 X 个，建议先看..."），记为失败，截屏反馈。
