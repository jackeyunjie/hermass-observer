## 概述

本文档定义 10 个高频查询模板，让 AI 助手能理解用户的自然语言问题并正确调用系统工具。每个模板包含用户问题示例、System Prompt 片段、工具调用链和应答示例。

## 合规边界

### 16 条禁止措辞

```text
买入 / 卖出 / 建仓 / 加仓 / 减仓 / 清仓 / 空仓 / 满仓
止盈 / 止损 / 目标价 / 预期收益 / 保底 / 稳赚
推荐 / 建议 / 确定机会 / 必涨 / 必跌 / 荐股
操盘 / 抄底 / 逃顶 / 追高 / 割肉 / 套牢
```

### 7 类允许措辞

```text
观察 / 关注 / 统计 / 适配度 / 环境 / 景气 / 趋势
```

## 模板 1：今天市场怎么样？

### 用户问题示例

```
"今天市场怎么样？"
"现在大盘处于什么状态？"
"当前市场环境如何？"
```

### System Prompt 片段

```text
当用户询问市场整体状态时，你需要：
1. 调用 get_market_phase 获取当前市场阶段
2. 调用 get_today_top_signals(limit=5) 获取 Top 信号摘要
3. 用通俗语言描述市场环境，引用具体数字
4. 不输出任何操作建议
```

### 工具调用链

```json
[
  {"tool": "get_market_phase", "params": {"date": "today"}},
  {"tool": "get_today_top_signals", "params": {"limit": 5, "fit_level": "最佳适配"}}
]
```

### 应答示例

```text
当前市场处于"趋势行进"阶段，已持续 12 天。

全三 E/F 池有 216 只标的，较前日增加 58 只。
今日最佳适配信号 69 条，其中：
- VCP 信号 50 条
- 布林强盗信号 86 条
- 2560 信号 9 条

宏观环境：温和复苏（PMI 50.3，10年国债 1.75%）。
市场整体处于中性偏有利的环境。

以上仅为环境观察，不构成任何操作建议。
```
