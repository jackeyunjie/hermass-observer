# State Display Terms Reference

## 目的

这份表述块只服务前台解释与 Coach 检索，不替代底层 `state_hex / state_score` 编码。

原则：

- 保留 raw state：`E/F/C/0/-C/...`
- 前台解释使用结构语义，不使用“牛/熊/强推/弱势”这类主观定性词
- 解释只说明结构，不给投资建议

## 常见状态基础表述

以下表述只定义单个 state 的结构含义。实际卡片回答还应结合 `MN1/W1/D1` 组合、持续天数和共振状态生成动态先验表达。

### `E`

- Alias：`扩张·趋势·突破·稳定`
- 固定说明：价格结构处于扩张与趋势延续状态，已经完成突破，但短周期波动仍相对可控。

### `F`

- Alias：`扩张·趋势·突破·活跃`
- 固定说明：结构已进入扩张趋势，并完成关键结构突破，同时波动和活跃度更高。
- 与 `E` 的区别：`E` 更偏突破后的稳定推进，`F` 更偏突破后的活跃推进，短周期节奏更容易先出现切换。

### `C`

- Alias：`扩张·趋势·未突破·稳定`
- 固定说明：结构已经进入扩张和趋势阶段，但还没有形成明确突破，整体仍处于确认过程中。

### `0`

- Alias：`收缩·无方向·未突破·稳定`
- 固定说明：结构仍偏收缩，方向性不强，也没有形成突破，通常更像等待或蓄势状态。

### `-C`

- Alias：`收缩·趋势·未突破·稳定（负向）`
- 固定说明：结构带有负向趋势特征，尚未形成向上的突破确认，整体偏防守和观察阶段。

## 前台展示规则

### Quick Card

- 只显示一行：
  - `State：E/E/F（三周期共振刚形成，先验上更偏向观察共振能否延续，而不是直接外推强度。）`
- 不展开单周期 alias
- 括号内优先使用动态先验表达；缺少持续性数据时再退回结构解读。

### Deep Card

- 显示：
  - raw state
  - 结构解读
  - state 持续性
- alias 作为补充，不替代 raw state

### Evidence Card

- 显示：
  - raw state
  - 结构解读
  - `MN1/W1/D1 Alias`
  - state 持续性

## Coach 检索建议

当用户问：

- `E/F 是什么意思`
- `E 和 F 有什么区别`
- `0 状态代表什么`

Coach 应优先引用本固定表述块，再结合当前个股的 `State：MN1/W1/D1` 与 `结构解读` 作解释。

不要单独把某个状态解释成交易建议或涨跌判断。

## 动态先验表达

Coach 或卡片生成时，应优先结合：

- 大周期 `MN1/W1` 是否已经进入 `E/F`
- 短周期 `D1` 是 `E` 还是 `F`
- `MN1/W1/D1` 各自强状态持续了多久
- 三周期共振持续了多久

示例：

| 场景 | 推荐表达 |
|------|----------|
| 三周期共振刚形成 | `三周期共振刚形成，先验上更偏向观察共振能否延续，而不是直接外推强度。` |
| 大周期持续较久，D1 刚转 F | `大周期保持扩张趋势，D1 处于突破后的活跃推进段，先验上更容易先出现短周期节奏切换。` |
| 大周期强，D1 未进入 E/F | `大周期已有支撑，但短周期尚未补齐，先验上更关注 D1 是否转入 E/F。` |
| 只有单周期 E/F | `当前只有单周期进入 E/F，先验上仍属于局部结构，需等待更多周期确认。` |
