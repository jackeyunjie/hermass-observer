## 负值状态

负值表示 position 方向为负（价格低于支撑位），幅度不变。

| hex | score | 含义 |
|-----|-------|------|
| -C | -12 | 负向趋势行进 |
| -E | -14 | 负向强势突破（破位确认） |
| -F | -15 | 负向强势突破+波动 |

## E/F 定义

```
E = 14 = 8+4+2+0（扩张+有趋势+突破+稳定）
F = 15 = 8+4+2+1（扩张+有趋势+突破+活跃）

ef_count = 三周期中 E 或 F 的数量（0-3）
负值 -E/-F 不算 E/F
```
