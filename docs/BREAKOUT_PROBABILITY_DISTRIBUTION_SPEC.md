# Breakout Probability Distribution Spec

## 1. 目标

本规范定义一套前台可解释的概率分布提示层，用于回答：

1. 当前更像真突破还是假突破
2. 当前天花板更高还是更低
3. 当前持续性更可能延续还是衰减
4. 什么时候更适合等待，什么时候可小额试错，什么时候可顺风跟踪
5. D1 支撑位在哪里，跌破后意味着什么

这个能力不是给出确定性结论，而是基于：

- `位置`
- `资金流`
- `板块/行业`
- `State`
- `SR 边界`

输出一个**概率分布提示**。

---

## 2. 定位

定位：

**Hermass State 底座上的突破质量解释层**

它不是：

- 新的买点生成器
- 单独的策略系统
- 收益承诺器

它是：

- 帮用户区分真突破与假突破风险
- 帮用户识别高天花板与低天花板
- 帮用户判断该等待、试错还是跟踪

---

## 3. 输入维度

### 3.1 位置维度

最少输入字段：

- `reference_close`
- `nearest_support` / `d1_sr_support`
- `nearest_resistance` / `d1_sr_resistance`
- `sr_boundary_direction`
- `sr_distance_pct`
- `lifecycle_stage`
- `d1_ef_duration`

解释：

- 位置决定赔率
- 离支撑太远，赔率下降
- 刚突破但离边界很近，假突破风险提高

### 3.2 结构维度

最少输入字段：

- `mn1_state`
- `w1_state`
- `d1_state`
- `ef_count`
- `strategy_environment_fit`
- `market_phase`

解释：

- 同样是突破，若只有 D1 强而 MN1/W1 不支持，持续性更弱
- `ef_count >= 2` 时，突破更有环境支撑

### 3.3 板块维度

最少输入字段：

- `industry_state`
- `sector_resonance`
- `industry_position`
- `top_industries / weak_industries`

解释：

- 板块集中共振时，真突破概率提高
- 板块退潮时，假突破概率提高

### 3.4 资金流维度

参考 `moneyflow_usage_research.md`，资金流只作为：

1. 确认
2. 背离复核
3. 高位分歧预警

最少输入字段：

- `active_net_5d`
- `big_order_net_5d`
- `positive_days_5d`
- `big_positive_days_5d`
- `latest_active_net`

解释：

- 强结构 + 资金流同向 -> 真突破概率提高
- 强结构 + 资金流背离 -> 假突破与高位分歧概率提高

---

## 4. 输出对象

前台输出不是一个分数，而是四组概率判断：

```json
{
  "breakout_quality": {
    "true_breakout_prob": "high",
    "false_breakout_prob": "medium",
    "reason": "三周期共振成立，行业顺风，但刚突破时间还短"
  },
  "ceiling_quality": {
    "high_ceiling_prob": "medium",
    "low_ceiling_prob": "high",
    "reason": "已明显远离支撑，短期剩余赔率下降"
  },
  "persistence_quality": {
    "continue_prob": "medium",
    "fade_prob": "medium",
    "reason": "板块未形成强共振，持续性仍需观察"
  },
  "action_guidance": {
    "mode": "小额试错",
    "reason": "环境支持但突破尚未充分站稳"
  }
}
```

---

## 5. 概率等级

前台只显示五档，不显示伪精确数字：

- `very_high`
- `high`
- `medium`
- `low`
- `very_low`

中文映射：

- `很高`
- `较高`
- `中等`
- `较低`
- `很低`

原因：

- 当前阶段数据还不足以支撑精确概率百分比
- 五档更稳，也更适合产品前台

---

## 6. 真突破 / 假突破规则

### 6.1 真突破概率提高条件

满足越多，`true_breakout_prob` 越高：

1. `ef_count >= 2`
2. `sr_boundary_direction = above_resistance`
3. `sr_distance_pct <= 0.03`
4. `d1_ef_duration <= 3`
5. `strategy_environment_fit in {最佳适配, 适配}`
6. 行业处于顺风或有板块共振
7. 资金流连续 3/5 天确认

### 6.2 假突破概率提高条件

满足越多，`false_breakout_prob` 越高：

1. 只有 D1 强，MN1/W1 不支持
2. `sr_boundary_direction = above_resistance` 但 1-3 天内未站稳
3. 价格已明显远离支撑，且无板块共振
4. 高位放量但资金流背离
5. 市场处于退潮或高离散阶段
6. 历史同环境下该策略表现偏弱

### 6.3 前台话术

不要说：

- “这是假的”
- “一定是真突破”

要说：

- `当前更像真突破候选，但仍需看 1-3 天站稳情况。`
- `当前假突破风险偏高，原因是大环境与板块承接不足。`

---

## 7. 高天花板 / 低天花板规则

### 7.1 高天花板

表示：

- 剩余赔率仍大
- 离支撑不远但向上空间还足
- 尚未进入明显高位透支段

提高条件：

1. `rr_ratio` 高
2. `upside_pct` 明显高于 `downside_pct`
3. 刚突破或推进早期
4. 行业仍处于扩散期而非衰竭期

### 7.2 低天花板

表示：

- 已走出主要波段
- 剩余赔率下降
- 更适合持有管理，不适合新开跟踪

提高条件：

1. 已明显远离支撑
2. `d1_ef_duration` 较长
3. `rr_ratio` 下滑
4. 行业开始分化退潮

### 7.3 前台话术

- `高天花板概率较高：当前仍处于早期推进段，剩余赔率相对充足。`
- `低天花板概率较高：当前更像高位延展，适合管理已有观察，不适合把它当成早期样本。`

---

## 8. 持续性规则

### 8.1 持续性提高条件

1. `MN1/W1/D1` 同向
2. `ef_count >= 2`
3. 行业共振存在
4. 资金流连续确认
5. 市场 phase 不是退潮

### 8.2 衰减概率提高条件

1. 仅 D1 强
2. 行业退潮
3. 资金流背离
4. 价格离支撑过远
5. 已进入高位延展末端

### 8.3 前台话术

- `持续性偏高：结构、行业和资金流暂时同向。`
- `持续性一般：结构仍在，但行业与资金流没有同时支持。`
- `持续性偏弱：当前更像高位延展，后续更容易进入衰减。`

---

## 9. 动作建议层

输出只给四档，不给具体仓位：

- `等待确认`
- `小额试错`
- `标准跟踪`
- `重点关注`

### 9.1 等待确认

适用：

- 刚突破
- 假突破风险偏高
- 大环境或行业未确认

### 9.2 小额试错

适用：

- 结构支持
- 但站稳时间还短
- 或板块支持未完全同步

### 9.3 标准跟踪

适用：

- 真突破概率较高
- 持续性中高
- D1 支撑清晰

### 9.4 重点关注

适用：

- 结构、板块、资金流三者同向
- `rr_ratio` 明显较高
- 仍处于高天花板阶段

---

## 10. D1 支撑位提醒

前台必须直接显示：

- `D1 支撑位`
- `当前距支撑位`
- `跌破后意味着什么`

示例：

```text
D1 支撑位：24.66
当前距离：+2.8%
提示：仍在支撑上方，可观察回踩；若跌破则从跟踪降为等待确认
```

如果支撑缺失：

```text
D1 支撑位暂缺，不能据此判断回踩质量
```

## 11. 与页面的连接

### 11.1 `/watchlist`

优先接入，展示：

- `D1 支撑提醒`
- `假突破风险`
- `阶段位置`
- `投入层级`

### 11.2 `/research`

接入：

- 真突破/假突破解释
- 持续性判断
- 高天花板/低天花板判断

### 11.3 `/market`

接入：

- 当前更适合等待还是试错
- 当前哪些行业更容易出现真突破

---

## 12. 资金流边界

资金流不用于：

- 单独生成买点
- 覆盖 State/SR/行业环境
- 输出收益承诺

资金流只用于：

1. 确认
2. 背离复核
3. 高位分歧预警
4. 持续性排序

---

## 13. 一句话结论

这个能力的本质不是“预测涨跌”，而是：

**把突破样本拆成真突破概率、假突破风险、天花板高低、持续性强弱，再结合 D1 支撑位与大环境给出等待/试错/跟踪的动作层提示。**
