# 美股 State 特征与 A 股对比分析框架

版本：v1.0
日期：2026-05-25
状态：**已归档**
关联文档：`docs/MULTICYCLE_STATE_STRATEGY_WHITEPAPER.md`、`docs/SYSTEM_ARCHITECTURE.md`

> 范围声明：本系统已专注 A 股市场。本文档仅作历史归档，记录美股对比分析框架的设计思路，不作为 A 股系统的设计依据。

---

## 概述

本文档定义美股 State 特征与 A 股的系统性对比分析框架。目标是：**在美股 State MVP 数据就绪后，有一套明确的分析维度、基准数据和统计方法，确保两个市场的对比有方向、可复现、可解释。**

### 适用条件

本框架在美股 State MVP 完成后启动。启动前提：

- 美股 Foundation DB 已生成（含 MN1/W1/D1 State、SR、ATR）
- 美股 State 缓存已生成（至少 100 个交易日）
- 至少一条策略信号模块已在美股数据上运行

### 分析原则

1. **数据驱动**：所有结论必须基于本地 Foundation DB 的统计输出，不依赖外部报告或直觉判断。
2. **口径对齐**：两个市场使用完全相同的 State 公式、4-bit 编码、E/F 定义和分析脚本。
3. **差异导向**：不追求证明"哪个市场更好"，而是识别两个市场在 State 空间中的结构性差异，为策略适配提供依据。
4. **先描述后解释**：先用数据描述差异是什么，再用市场结构解释为什么。

## 第 1 章：对比维度定义

### 1.1 六个核心维度

| 序号 | 维度 | 定义 | 关键问题 |
|------|------|------|----------|
| D1 | 收缩期特征 | base=0 状态的持续时间、深度、恢复模式 | 美股收缩期是否更短更浅？收缩后释放路径是否更可靠？ |
| D2 | 突破模式 | position_bit 从 0 转 2 的触发条件和后续表现 | 美股突破是否更持久？假突破比例是否不同？ |
| D3 | 趋势持续性 | E/F 状态的持续天数、退出原因、重新进入概率 | 美股趋势是否更长？退出模式是否不同？ |
| D4 | 板块效应 | 行业/板块间的 State 同步性和传导速度 | 美股板块联动是否更强？传导是否更快？ |
| D5 | 政策/宏观冲击 | 外部事件对 State 分布的冲击幅度和恢复速度 | 美股对宏观数据的反应速度和恢复模式？ |
| D6 | 成交量特征 | 成交量与 State 转换的关系、量价配合模式 | 美股成交量是否更平滑？量价关系是否不同？ |

### 1.2 维度间关系

```text
收缩期特征 (D1)
    │
    ▼
突破模式 (D2) ──→ 趋势持续性 (D3)
    │                    │
    ▼                    ▼
板块效应 (D4)     政策/宏观冲击 (D5)
    │                    │
    └────────┬───────────┘
             ▼
      成交量特征 (D6) ── 贯穿所有维度
```

## 第 2 章：各维度 A 股已知规律

以下规律均来自本项目 Foundation DB 的本地验证数据（2025-06-01 至 2026-05-01，223 个交易日，约 5,000 只 A 股）。

### 2.1 D1 收缩期特征

**数据来源**：`outputs/project/vcp_optimal_state_search.md`、`outputs/project/state_transition_analysis.md`

| 观察项 | A 股数据 | 含义 |
|--------|----------|------|
| D1 近 5 日收缩后释放样本 | 1,473 条 | 占全部信号的 3.4% |
| D1 近 10 日收缩后释放样本 | 2,637 条 | 占全部信号的 6.1% |
| D1 近 20 日收缩后释放样本 | 5,243 条 | 占全部信号的 12.1% |
| 收缩后释放 20d 超额 | +1.66%（t-stat=3.98） | 路径条件有效区分信号质量 |
| 无收缩前兆的扩张 20d 超额 | +0.45% | 弱于路径组 |
| 收缩后释放 vs 无收缩前兆 | +1.21% 差异 | 路径条件有实际增量价值 |

**A 股特征总结**：

- A 股收缩期通常较短（5-20 日即出现有效释放信号），但收缩幅度可能较深（受涨跌停板放大恐慌情绪）。
- 收缩后释放路径在 A 股有显著超额收益，且时间窗口越窄（5 日 vs 20 日），超额越高，说明 A 股的收缩释放具有"时效性"——越新鲜的释放信号越有价值。
- A 股的收缩往往伴随成交量急剧萎缩（筹码锁定），释放时伴随放量突破。

### 2.2 D2 突破模式

**数据来源**：`outputs/project/2560_state_market_match.md`、`outputs/project/vcp_optimal_state_search.md`

| 观察项 | A 股数据 | 含义 |
|--------|----------|------|
| 2560 Top20 平均收益 | +13.7%（最高 +19.1%） | 突破后趋势延续能力强 |
| 2560 突破确认条件 | VOL5 > VOL60（20/20）、MA25 上行（17/20） | 量价配合是突破确认的核心 |
| VCP E/E/F 组合 20d 超额 | +17.02%（63.64% 胜率） | 三周期共振突破表现最强 |
| VCP E/E/E 组合 20d 超额 | +14.74%（68.18% 胜率） | 波动稳定态突破更可靠 |
| 2560 市场匹配 full_match | 62 条 / 167 条有效推荐 | 只有 37% 的推荐有完整的行业市场支持 |

**A 股特征总结**：

- A 股突破需要明确的量价配合确认，纯价格突破假突破率高。
- 三周期 E/F 共振是突破质量的强力过滤器，E/E/F 组合显著优于其他组合。
- A 股突破后的趋势延续依赖板块联动——仅有 37% 的推荐有完整的行业 ETF 支持，说明 A 股板块效应分化严重。
- 涨停板制度导致部分"突破"实际不可执行（涨停封板无法买入），需要可交易性过滤。

### 2.3 D3 趋势持续性

**数据来源**：`outputs/project/state_transition_analysis.md`（8,370,935 条 D1 跃迁事件）

| 观察项 | A 股数据 | 含义 |
|--------|----------|------|
| B->B（扩张无方向→扩张无方向） | 10,879 条，20d 超额 -1.87% | 最常见跃迁，无趋势延续 |
| B->F（扩张无方向→突破活跃） | 4,206 条，20d 超额 -1.47% | 突破后波动活跃反而负超额 |
| F->B（突破活跃→扩张无方向） | 1,580 条，20d 超额 -1.92% | 趋势退出后表现差 |
| B->9（扩张无方向→收缩有方向） | 4,668 条，20d 超额 -1.52% | 从扩张退回收缩 |
| -F->C（负突破活跃→正扩张有方向未突破） | 555 条，20d 超额 +1.75% | 底部反转信号较强 |

**A 股特征总结**：

- A 股 E/F 状态持续时间相对较短——F->B（趋势退出）频繁出现，说明 A 股趋势容易被外部事件打断。
- A 股的"假突破"问题突出——B->F（突然进入波动活跃态）后 20d 超额为负，说明很多突破是短暂的。
- A 股底部反转信号（-F->C）有正超额，说明 A 股的"否极泰来"模式有一定规律性。
- A 股趋势持续性受制度因素影响：T+1 限制了日内止损能力，涨跌停板放大了趋势中断的冲击。

### 2.4 D4 板块效应

**数据来源**：`docs/MA2560_STATE_MARKET_MATCH_RULE.md`、`config/industry_rotation_assets.json`

| 观察项 | A 股数据 | 含义 |
|--------|----------|------|
| 2560 full_match（有行业支持） | 62 条，超额显著高于 market_unsupported | 板块联动是信号质量的关键因子 |
| 2560 market_unsupported（无行业支持） | 97 条 | 超过半数推荐缺乏行业支持 |
| 行业 ETF ef_count >= 2 阈值 | 用于过滤板块支持强度 | 板块需要至少 2 个周期处于 E/F 才算有效支持 |
| 行业轮动特征 | A 股行业轮动快，题材一日游频繁 | 板块效应持续时间短 |

**A 股特征总结**：

- A 股板块效应高度分化——少数行业独强时个股信号更可靠，行业分散时信号质量下降。
- A 股行业轮动速度快，"题材一日游"现象频繁，板块效应的持续性弱于美股。
- A 股的板块联动更多由政策驱动（如新能源补贴、半导体国产替代），而非基本面驱动。

### 2.5 D5 政策/宏观冲击

**数据来源**：系统设计文档中的 A 股制度边界定义

| 观察项 | A 股特征 | 含义 |
|--------|----------|------|
| T+1 制度 | 信号确认与执行之间有滑点风险 | 冲击后恢复路径更曲折 |
| 涨跌停板 | 10%/20% 涨跌停限制 | 极端冲击时 State 可能"卡"在边界态 |
| ST/停牌 | 个股可能突然停牌 | State 计算中断 |
| 政策驱动 | 国务院/证监会政策对市场冲击大 | 政策冲击后 State 分布可能发生结构性变化 |
| 北向资金 | 外资流向对 A 股有边际影响 | 宏观冲击通过北向资金传导 |

**A 股特征总结**：

- A 股的宏观冲击主要通过政策渠道传导（降准降息、行业政策、监管措施），而非纯市场力量。
- 涨跌停板制度导致极端冲击时 State 分布出现"截断"——部分股票的状态被锁死在边界值。
- A 股的冲击恢复通常需要政策面配合，纯技术面恢复能力弱于美股。

### 2.6 D6 成交量特征

**数据来源**：`docs/STRATEGY_DEFINITIONS.md`（2560 策略的量能规则）

| 观察项 | A 股数据 | 含义 |
|--------|----------|------|
| 2560 VOL5 > VOL60 命中率 | 20/20（100%） | 短期放量是 A 股突破的必要条件 |
| VCP 收缩期成交量 | 逐步萎缩，筹码锁定 | 量缩价稳是收缩有效的标志 |
| VCP 释放期成交量 | 显著放大 | 放量突破是信号确认的核心 |
| A 股散户占比高 | 成交量波动大，情绪驱动明显 | 量价关系的噪声更大 |

**A 股特征总结**：

- A 股成交量是突破确认的硬性条件——2560 策略中 VOL5 > VOL60 命中率 100%。
- A 股成交量波动幅度大，受散户情绪和游资炒作影响，日内成交量可能剧烈波动。
- A 股的量价配合模式更"极端"——缩量收缩时极度萎缩，放量突破时极度放大。

## 第 3 章：美股预期差异分析

以下差异基于市场结构推断，需在美股 State MVP 数据就绪后用实际数据验证。

### 3.1 D1 收缩期特征 — 美股预期

| 推断维度 | A 股表现 | 美股预期 | 推断依据 |
|----------|----------|----------|----------|
| 收缩期时长 | 较短（5-20 日有效释放） | 可能更长 | 美股机构持仓比例高，筹码锁定需要更长时间确认 |
| 收缩幅度 | 受涨跌停板放大 | 可能更温和 | 无涨跌停板，价格调整更连续 |
| 收缩后释放可靠性 | 20d 超额 +1.66%（t-stat=3.98） | 可能更可靠 | 机构主导的突破通常更有持续性 |
| 成交量萎缩程度 | 极度萎缩 | 可能更平滑 | 做市商和算法交易维持基础流动性 |

**待验证假设**：

- H1: 美股收缩后释放路径的 t-stat 高于 A 股（机构主导的突破更可靠）
- H2: 美股收缩期中位数长于 A 股（筹码锁定确认需要更长时间）
- H3: 美股收缩期波动率分布更集中（无涨跌停板，无极端值截断）

### 3.2 D2 突破模式 — 美股预期

| 推断维度 | A 股表现 | 美股预期 | 推断依据 |
|----------|----------|----------|----------|
| 量价配合必要性 | VOL5 > VOL60 命中率 100% | 可能降低 | 美股流动性充足，突破不一定需要放量确认 |
| 假突破比例 | 较高（B->F 20d 超额为负） | 可能更低 | 机构主导的突破更有方向性 |
| 突破执行可行性 | 涨停封板导致部分突破不可执行 | 几乎全部可执行 | 无涨跌停板，T+0 可随时入场 |
| 三周期共振过滤 | E/E/F 20d 超额 +17.02% | 仍应有效 | 多周期共振是普适的技术分析原理 |

**待验证假设**：

- H4: 美股突破不需要 VOL5 > VOL60 作为硬性条件（流动性自维持）
- H5: 美股 E/E/F 组合的超额收益高于 A 股（突破更可靠，趋势延续更长）
- H6: 美股无"涨停不可执行"问题，full_match 比例应高于 A 股的 37%

### 3.3 D3 趋势持续性 — 美股预期

| 推断维度 | A 股表现 | 美股预期 | 推断依据 |
|----------|----------|----------|----------|
| E/F 持续天数 | 较短，F->B 频繁 | 可能更长 | 机构持仓稳定，趋势不易被散户情绪打断 |
| 趋势退出模式 | F->B（波动收敛退出）为主 | 可能更渐进 | 做市商缓冲导致退出更平滑 |
| 底部反转信号 | -F->C 20d 超额 +1.75% | 可能更弱 | 美股做空机制完善，反转信号更早被捕获 |
| 趋势加速段 | 布林强盗 volatility_bit=0 更稳 | 可能不同 | 美股波动率特征不同，volatility_bit 阈值可能需要调整 |

**待验证假设**：

- H7: 美股 E/F 状态中位持续天数 > A 股
- H8: 美股趋势退出更多由基本面驱动（财报、指引），A 股更多由政策/情绪驱动
- H9: 美股 -F->C 底部反转信号的超额收益弱于 A 股（做空机制更完善）

### 3.4 D4 板块效应 — 美股预期

| 推断维度 | A 股表现 | 美股预期 | 推断依据 |
|----------|----------|----------|----------|
| 板块联动强度 | 分化严重，full_match 37% | 可能更强 | 美股行业 ETF 更成熟，机构配置更系统化 |
| 板块轮动速度 | 快，题材一日游 | 可能更慢 | 机构持仓周期长，板块趋势更持久 |
| 板块驱动因素 | 政策驱动为主 | 基本面驱动为主 | 美股市场定价效率更高 |
| 传导机制 | 政策→行业→个股 | 宏观→行业→个股 | 美股更多通过盈利预期和估值传导 |

**待验证假设**：

- H10: 美股行业 ETF 的 ef_count 与个股信号质量的相关性高于 A 股
- H11: 美股板块效应的持续时间（中位数）长于 A 股
- H12: 美股不需要 A 股式的"行业 ETF ef_count >= 2"过滤，或阈值可降低

### 3.5 D5 政策/宏观冲击 — 美股预期

| 推断维度 | A 股表现 | 美股预期 | 推断依据 |
|----------|----------|----------|----------|
| 冲击传导速度 | 政策传导，有一定滞后 | 更快 | 美股对宏观数据（非农、CPI、FOMC）反应即时 |
| 冲击恢复模式 | 需要政策面配合 | 更市场化 | 价格发现效率更高，技术面恢复能力更强 |
| State 分布冲击幅度 | 涨跌停板导致截断 | 更连续 | 无涨跌停板，价格调整更平滑 |
| 冲击类型 | 政策驱动为主 | 宏观数据+地缘政治+FOMC | 冲击来源更多元 |

**待验证假设**：

- H13: 美股 FOMC/CPI 日前后 State 分布变化幅度可量化，且恢复速度快于 A 股政策冲击
- H14: 美股 State 分布无"截断"效应，分布形态更接近正态
- H15: 美股财报季（每季度）对个股 State 的冲击大于 A 股（A 股财报披露更分散）

### 3.6 D6 成交量特征 — 美股预期

| 推断维度 | A 股表现 | 美股预期 | 推断依据 |
|----------|----------|----------|----------|
| 基础流动性 | 个股差异大，小盘股流动性差 | 更均匀 | 做市商制度保障基础流动性 |
| 成交量波动 | 散户情绪驱动，波动大 | 更平滑 | 算法交易和做市商缓冲 |
| 量价关系 | VOL5 > VOL60 是硬性条件 | 可能不同 | 美股突破不一定需要放量确认 |
| 日内成交量分布 | 集中在开盘和收盘 | 更均匀 | 盘前盘后也有流动性 |

**待验证假设**：

- H16: 美股 VOL5 > VOL60 条件的信号区分度弱于 A 股
- H17: 美股成交量与 State 转换的相关性系数低于 A 股
- H18: 美股可用成交量加权的 State 计算（因流动性更均匀），A 股不适合

## 第 5 章：统计方法

### 5.1 对比统计指标

每个维度的对比分析统一使用以下指标体系：

| 指标 | 公式 | 用途 |
|------|------|------|
| 平均超额收益 | mean(forward_return - benchmark_return) | 核心效果度量 |
| 95% 置信区间 | bootstrap CI (10,000 resamples) | 不确定性度量 |
| 胜率 | P(forward_return > benchmark_return) | 方向一致性 |
| 盈亏比 | mean(positive_returns) / abs(mean(negative_returns)) | 风险收益对称性 |
| t-stat | mean_excess / (std / sqrt(n)) | 统计显著性 |
| 样本量 | n | 结论可信度 |

### 5.2 跨市场可比性指标

以下指标用于直接比较两个市场，消除绝对值差异：

| 指标 | 定义 | 用途 |
|------|------|------|
| 收缩比 | base=0 天数 / 总交易日数 | 收缩期占比 |
| 释放效率 | 收缩后释放 20d 超额 / 无收缩前兆 20d 超额 | 收缩路径的增量价值 |
| 趋势持续中位数 | E/F 状态连续天数的中位数 | 趋势长度 |
| 突破可靠性 | position_bit 0→2 后 20d 超额 > 0 的比例 | 突破成功率 |
| 板块支持率 | 有行业 ef_count >= 2 的信号占比 | 板块效应覆盖 |
| State 分布熵 | -sum(p_i * log(p_i))，p_i 为各 State 占比 | 市场状态多样性 |
| 跃迁集中度 | Top10 跃迁类型占总跃迁的比例 | 状态转移的可预测性 |

### 5.3 分析脚本复用

以下 A 股已有脚本可直接复用于美股（输入改为美股 Foundation DB）：

### 6.2 分析步骤

```text
步骤 1: 描述性统计（1-2 天）

  ├── 收缩期特征: base=0 持续天数、收缩后释放路径验证
  ├── 突破模式: position_bit 0→2 后收益统计
  ├── 趋势持续性: E/F 连续天数分布
  └── 输出: outputs/us_vs_cn/trend_structure_comparison.json

步骤 3: 维度 D4-D6 分析（2-3 天）
  ├── 板块效应: 行业 ETF ef_count 与个股信号相关性

  ├── 成交量特征: 量价关系跨市场对比
  └── 输出: outputs/us_vs_cn/market_structure_comparison.json

步骤 4: 策略适配差异分析（2-3 天）
  ├── 三策略在美股的 State 组合搜索
  ├── 最优组合与 A 股对比
  ├── 适配度评分的有效性跨市场验证
  └── 输出: outputs/us_vs_cn/strategy_fit_comparison.json

步骤 5: 综合报告（1 天）
  ├── 汇总六个维度的发现
  ├── 美股 State 参数是否需要调整的建议
  ├── 策略规则是否需要市场特定调整的建议
  └── 输出: outputs/us_vs_cn/cross_market_analysis_report.md
```
