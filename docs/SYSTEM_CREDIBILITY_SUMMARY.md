# 系统可信度摘要

版本：v1.0 | 日期：2026-05-24

> 范围声明：本文档用于说明 A 股活跃生产系统的可信度。创始人跨市场类比研究不再作为当前系统可信度主论据；相关内容如需保留，应视为历史研究参考。

---

## 一句话定位

**Hermass Observer 是一个 A 股多周期 State 环境识别系统** — 它不预测涨跌，只回答"当前市场环境适合哪个策略"。

当前系统可信度主来源仅保留 A 股本地可复现证据：

| 证据类型 | 说明 | 来源 |

## 系统边界声明

| 边界 | 说明 |
|------|------|
| 底座只读 | State 公式不可修改，D1 视角天条不可改变 |
| 策略不重写 | 策略信号账本只记录事实，不简化规则 |
| 提醒不越界 | 不输出买入/卖出/加仓/减仓/荐股等操作建议 |
| 统计有边界 | 所有超额收益和胜率均为历史回溯，不代表未来表现 |
| 校准待完成 | 前向观察样本积累中，预计 2026-Q4 首次校准 |
