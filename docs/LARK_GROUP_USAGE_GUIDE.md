# 飞书群使用指南

适用对象：内测用户
日期：2026-05-28

---

## 如何使用

在飞书群中 @Hermass Bot，然后输入你的问题。Bot 会自动识别你的意图并回复。

## 常用问题示例

### 了解市场

```
@Hermass Bot 今天市场怎么样
```
→ 返回市场阶段、E/F 池规模、宏观环境、板块共振情况

```
@Hermass Bot 哪些行业在动
```
→ 返回今日板块共振信号（哪些行业正在集体启动）

### 查询个股

```
@Hermass Bot 002049 怎么样
```
→ 返回该股票的 State 环境、行业景气、财务趋势、主要风险

```
@Hermass Bot 紫光国微深度分析
```
→ 返回更详细的深度研究卡，含商业模式、财务同比、估值参考

### 了解策略

```
@Hermass Bot VCP 最近表现怎么样
```
→ 返回 VCP 策略的回测结果（收益、胜率、最大回撤）

```
@Hermass Bot 三个策略对比
```
→ 返回 VCP/2560/布林强盗的对比表格

### 学习系统

```
@Hermass Bot 什么是 State 的 E/F
```
→ 返回 E/F 状态的结构解读

```
@Hermass Bot /coach 2560 策略怎么理解
```
→ 返回 2560 策略的核心逻辑和适用环境

### 了解宏观

```
@Hermass Bot 宏观环境适合交易吗
```
→ 返回宏观象限、四维评分、对各策略的影响

```
@Hermass Bot 帮我查一下 002049 的估值
```
→ 返回 PE/PB 数据及行业对比（仅供参考）

### 其他

```
@Hermass Bot 帮助
```
→ 返回完整的功能清单

---

## 注意事项

- Bot 的所有回答**仅供研究参考，不构成投资建议**
- 历史统计数据不代表未来表现
- "待校准"表示该统计数据正在积累验证样本，不是缺陷
- Bot 不会推荐买入或卖出任何股票
