## 这个系统是什么？

Hermass Observer 是一个 A 股市场状态观察系统。它每天自动扫描全市场股票，用数学方法判断每只股票处于趋势生命周期的哪个阶段，然后告诉您哪些股票的市场环境与某个经典策略最匹配。

### 系统做什么

- 每天计算全市场股票在月线、周线、日线三个时间维度的状态
- 判断每只股票处于趋势的哪个阶段（新生、行进、延展）
- 用三个经典策略分别扫描全市场，记录策略信号
- 为每个信号标注"环境适配度"——这个信号在当前市场环境下的匹配程度
- 生成每日报告和策略提醒

### 系统不做什么

- 不预测涨跌
- 不给出买入、卖出、加仓、减仓的建议
- 不推荐具体股票
- 不管理您的账户或资金
- 不承诺任何收益

**一句话总结**：系统是一个市场环境观察工具，帮助您理解"当前市场处于什么状态"和"哪些策略适合当前环境"，但所有投资决策需要您自己做出。

## 每日总报怎么看？

每天系统会生成一份总报（`daily_research_brief`），打开 HTML 文件即可在浏览器中查看。总报包含以下模块：

### 模块 1：今日概览

总报最上方的统计卡片，提供当日市场的快速扫描：

| 指标 | 含义 | 怎么看 |
|------|------|--------|
| 全三 E/F 池 | 月线、周线、日线同时处于 E 或 F 状态的股票数量 | 数字越大，市场整体越强。200+ 代表较强的市场环境 |
| 最佳适配信号 | 在当前环境下与策略"最佳匹配"的信号数量 | 关注这些信号优先级最高 |
| 各策略信号分布 | VCP、2560、布林强盗各产生了多少信号 | 了解哪个策略在当前环境下最活跃 |
| 宏观先验 | 宏观环境评分（0-10）和数据状态 | 高分有利，低分谨慎，数据不足时标注"partial" |
| 风险偏好 | 市场风格偏向（成长/小盘/中性） | 帮助理解当前资金偏好 |

### 模块 2：最佳适配信号聚焦表

这是总报最核心的部分。表格展示当日与策略环境"最佳匹配"的信号，每行包含：

| 列名 | 含义 | 怎么用 |
|------|------|--------|
| 股票 | 股票代码和名称 | — |
| 行业 | 申万一级行业 | 了解所属产业链 |
| 策略 | 触发信号的策略名称 | VCP / 2560 / 布林强盗 |
| 生命周期 | 该股票当前处于趋势的哪个阶段 | 新生/行进/延展 |
| 适配理由 | 为什么被评为"最佳适配" | 重点关注这个列的解释 |
| State 环境 | 月线/周线/日线的状态码 | 三个 E 或 F 最强 |
| 本地验证 | 历史统计数字 | 含样本量和超额收益 |

### 模块 3：2560 市场匹配分组

专门针对 2560 策略的分组展示，按匹配等级分为四组：

| 分组 | 含义 | 关注度 |
|------|------|--------|
| full_match | 个股规则成立 + State 匹配 + 行业 ETF 共振 | 最高 — 三重确认 |
| stock_only | 个股规则和 State 成立，但行业 ETF 数据缺失 | 次高 — 缺市场共振确认 |
| market_unsupported | 个股规则和 State 成立，但行业 ETF 不支持 | 谨慎 — 市场环境不配合 |
| not_match | 个股不满足 2560 的适配 State 组合 | 不关注 |

### 模块 4：校准状态

报告中会显示"校准状态"字段，当前值为"待校准"（calibration_not_available）。

**"待校准"是什么意思？**

系统的适配度评分是基于规则映射的（比如"VCP 在趋势新生阶段 = 最佳适配"）。要验证这个映射是否真的有效，需要积累足够多的历史数据，比较"最佳适配"信号和"不适配"信号的未来收益差异。

当系统积累了足够样本（通常需要 3-6 个月的前向观察数据），校准层会自动验证适配度排序的有效性。校准通过后，"待校准"会变为"已校准"，并附带具体的统计数字。

**校准前，适配度还有参考价值吗？** 有。适配度基于经典策略理论和 State 环境的逻辑匹配，即使未经历史数据校准，其逻辑基础是成立的。但校准后您可以看到具体的收益统计，参考价值更高。

## 三个策略分别是什么？

系统跟踪三个经典趋势策略，分别对应趋势生命周期的不同阶段。

### VCP（波动收缩形态）

**一句话**：当市场从沉睡中苏醒、波动收缩到极致后突然放量突破时，VCP 捕捉这个"弹簧释放"的瞬间。

**适合的环境**：
- 趋势新生阶段（系统标注为"新生"）
- 日线刚刚从收缩状态脱离
- 三周期共振刚形成（不超过 5 天）

**本地验证结论**：日线近 20 日经历收缩后释放的样本，20 日平均超额收益 +4.69%，胜率 56.16%。

**在总报中的特征**：适配理由通常包含"D1 刚脱离收缩"、"三周期共振新近形成"、"VCP 最佳适配"。

### 2560（均线回踩确认）

**一句话**：当趋势已经确认、股价回到 25 日均线附近时，用成交量确认回调质量，在趋势行进中寻找节奏。

**适合的环境**：
- 趋势行进阶段（系统标注为"行进"）
- 25 日均线坚定向上
- 所在行业的 ETF 也处于强势状态（full_match）

**本地验证结论**：State 组合 E/E/F、E/F/F、E/F/E 配合行业 ETF 共振（ef_count >= 2）时，信号质量最高。

**在总报中的特征**：适配理由通常包含"波动稳定"、"ef_count=3"，匹配等级为 full_match 的信号最值得关注。

### 布林强盗（波动突破跟踪）

**一句话**：当价格突破布林带上轨、确认趋势进入加速阶段时，布林强盗捕捉波动扩张带来的趋势延续。

**适合的环境**：
- 趋势延展阶段（系统标注为"延展"）
- 价格突破布林上轨
- 波动从稳定转为活跃的转折点

**一个违反直觉的发现**：本地数据显示，布林强盗信号在波动稳定时（volatility_bit=0）表现更好（+0.59%），波动活跃时反而更差（-0.49%）。这意味着信号的最佳环境是"刚从稳定进入波动"的转折点，而不是"已经波动很大"的时候。

**在总报中的特征**：适配理由包含"波动稳定"时更值得关注，包含"波动偏活跃"时要谨慎。

## 策略提醒怎么看？

策略提醒（`strategy_reminder`）是比总报更详细的信号清单，每条提醒包含以下信息：

### 提醒卡片字段解释

| 字段 | 含义 | 示例 |
|------|------|------|
| 生命周期 | 趋势阶段 | 新生 / 行进 / 延展 |
| 策略信号 | 触发的具体信号名称 | VCP突破确认 / 2560金叉 / 布林强盗多头触发 |
| 环境适配 | 信号与当前环境的匹配程度 | 最佳适配 / 适配 / 弱适配 / 待观察 |
| State 环境 | 三周期状态 | E/E/E 表示月/周/日线全部为 E 状态 |
| 持续 | D1 E/F 持续天数和三周期共振持续天数 | D1 5 / all-three 2 |
| SR 位置 | 价格与支撑/阻力位的关系 | above_resistance 表示价格在阻力位上方 |
| 环境标签 | 当前环境的特征描述 | 波动稳定 / 三周期共振新近形成 / D1收缩充分 |
| iFinD 摘要 | 基本面质量标签 | 质量健康 / 现金流健康 / 盈利质量中性 |
| 证据档 | 证据评级 | A（强证据）/ B（中等）/ C（弱） |
| 统计 | 历史验证数据或"待校准" | 待校准 / 具体百分比 |

### 常见环境标签解释

| 标签 | 含义 | 对策略的影响 |
|------|------|-------------|
| 三周期共振新近形成 | 月/周/日线同时为 E 或 F 的状态刚形成不久（≤5 天） | VCP 的最佳环境之一 |
| D1 刚脱离收缩 | 日线从收缩态进入扩张态，不超过 3 天 | VCP 的核心路径条件 |
| D1 收缩充分 | 日线经历了较长时间的收缩 | 收缩越充分，后续释放力度可能越大 |
| 波动稳定 | 日线 volatility_bit=0 | 2560 和布林强盗的偏好环境 |
| 波动偏活跃 | 日线 volatility_bit=1 | 布林强盗的信号已触发但要谨慎 |
| 价格位于阻力区间上方 | D1 收盘价高于日线阻力位 | 可能已进入趋势延展 |
| W1 刚脱离收缩 | 周线从收缩态进入扩张态 | 中期趋势转强信号 |
| full_match（仅 2560） | 个股规则 + State 匹配 + 行业 ETF 共振 | 2560 的最高质量确认 |
| not_match（仅 2560） | 不满足 2560 的适配 State 组合 | 2560 信号不建议关注 |

## 适配度等级说明

系统为每个策略信号标注五个适配度等级：

| 等级 | 含义 | 建议关注度 |
|------|------|-----------|
| 最佳适配 | 策略在当前环境下处于最佳生命周期阶段 | 优先关注 |
| 适配 | 策略在当前环境下有历史支撑 | 次要关注 |
| 弱适配 | 策略可运行但历史上该环境表现较弱 | 谨慎参考 |
| 待观察 | 缺乏足够数据判断环境匹配度 | 暂不参考 |
| 不适配 | 当前环境明确不适合该策略 | 不参考 |

**各策略的最佳适配环境**：

| 策略 | 最佳适配 | 适配 | 弱适配 |
|------|----------|------|--------|
| VCP | 新生 | — | 行进、延展 |
| 2560 | 行进 | 新生 | 延展 |
| 布林强盗 | 延展 | 行进 | 新生 |

## 本地验证数据怎么理解？

总报和提醒中经常出现本地验证数据，格式如：

```text
本地验证：近20日D1收缩后释放路径，20日平均超额+1.67%，验证区间2025-06-01至2026-05-01，样本43259。
```

**逐项解释**：

| 字段 | 含义 |
|------|------|
| 近20日D1收缩后释放路径 | 验证的假设条件 |
| 20日平均超额+1.67% | 满足该条件的信号，未来 20 日的平均超额收益（相对全市场等权） |
| 验证区间 2025-06-01 至 2026-05-01 | 使用的历史数据范围 |
| 样本 43259 | 参与统计的信号数量 |

**注意**：
- 超额收益是历史统计平均值，不代表每个信号都会达到这个水平
- 样本量越大，统计越可靠
- "待校准"表示该统计数据尚未通过系统的校准流程

## 常见问题 FAQ

### Q1：系统推荐我买什么股票？

A：系统不推荐任何股票。系统只展示策略信号和环境适配度，所有投资决策需要您独立做出。

### Q2："最佳适配"的信号一定会涨吗？

A：不一定。"最佳适配"表示策略在当前环境下处于最佳生命周期阶段，历史上该环境的平均表现较好。但每个具体信号的结果可能与平均值有较大偏差。

### Q3：三个策略同时发出信号怎么办？

A：说明该标的可能处于多个阶段的交汇点。系统会分别标注每个信号的适配度，您可优先关注适配度最高的那个。

### Q4：为什么有些信号标注"弱适配"但还是出现在提醒中？

A：提醒层展示所有触发的信号，适配度只是一种环境参考。标注"弱适配"是提醒您该信号在当前环境下历史上表现较弱，需要更谨慎地判断。

### Q5："待校准"什么时候会变成真实统计？

A：系统每天记录信号和未来收益标签。当某个策略×环境组合积累到足够样本（通常需要 3-6 个月），校准流程会自动验证适配度排序是否与实际收益一致。校准通过后，"待校准"会变为具体的统计数字。

### Q6：布林强盗在波动活跃时不是应该更好吗？为什么数据显示更差？

A：这是本地数据的一个有趣发现。布林强盗的信号本身是"波动突破"，但如果触发时波动已经很活跃，说明突破可能已经走了很远，追高的风险更大。在波动稳定时触发，说明这是突破的初始阶段，历史表现更稳。

### Q7：2560 策略为什么需要行业 ETF 共振？

A：2560 是趋势中的回踩策略。如果只有个股强而行业弱，回踩可能是趋势反转的开始。行业 ETF 共振（full_match）确认了"大环境也在趋势中"，提高了回踩后继续上行的概率。

### Q8：State 环境中的 E/F 是什么意思？

A：E 和 F 是系统中最强的两种状态。E = 扩张 + 有趋势 + 突破确认 + 波动稳定（得分 14），F = 扩张 + 有趋势 + 突破确认 + 波动活跃（得分 15）。当月线、周线、日线同时为 E 或 F 时，称为"三周期共振"，是系统关注的最强信号。

### Q9：这份报告适合短线交易吗？

A：不适合。三个策略都是日线及以上级别的趋势策略，观察窗口为 5-60 个交易日。它们关注的是趋势的阶段特征，不是日内的价格波动。

### Q10：数据不足时系统会怎么处理？

A：系统在数据不足时会明确标注（如"宏观数据暂缺"、"产业链数据暂缺"），不会用默认值伪装为事实。标注"待观察"的信号就是缺乏足够环境数据来判断适配度的情况。

### Q11：我可以直接用这份报告做交易吗？

A：不建议。报告是研究工具，不是交易系统。如果您决定参考报告中的信息，建议：(1) 结合自己的判断，(2) 注意适配度等级，优先关注"最佳适配"，(3) 了解本地验证数据的含义和局限性，(4) 做好风险管理。

### Q12：报告中的"证据档 A/B/C"是什么意思？

A：证据档是对信号质量的综合评级，考虑了 State 环境、基本面质量、产业链景气等多个维度。A 档证据最充分，C 档证据较弱。这是一个研究参考指标，不代表信号的确定性。
