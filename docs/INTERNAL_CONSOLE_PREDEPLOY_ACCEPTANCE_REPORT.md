## 4. 非阻塞遗留项

以下问题当前不构成“不能部署”的阻塞项，但部署前最好再做一次人工复核。

### 4.1 浏览器本地标签页异常

现象：

- 在当前本机 Chrome 的旧标签页/重复新标签页访问 `/?mode=direction` 时，曾多次看到 `Internal Server Error`

但同时存在以下事实：

- `python3 -m py_compile web/main.py` 通过
- `FastAPI TestClient` 直接请求 `GET /?mode=direction` 返回 `200`
- 首页上下文函数 `_view_mode_summary('direction')` 与 `_quant_summary()` 已能正确返回最新结构

判断：

- 当前更像本地浏览器旧会话/本地 localhost 访问状态异常
- 不是首页模板或路由代码仍然确定性报错

处理建议：

- 部署前使用干净浏览器窗口或重启浏览器后再做一次人工复核
- 部署到服务器域名后重新完整走一遍主路径

### 4.2 首页残留口径的最终人工确认

已修复内容：

- 方向模式首页判断已从“最适配策略”改为“更适配的周期走势”

当前建议：

- 在干净浏览器会话中确认这些文案已实际渲染到页面

### 4.3 主动提醒账本缺失

现状：

判断：
