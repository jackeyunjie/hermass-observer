## 3. Test 7 回归 — 连续对话记忆

### 测试步骤

1. **清除之前的会话状态**（刷新页面，或关闭重开观象浮窗，确保新会话）
2. 第 1 轮：输入 `"000021 怎么样"`
3. 等待回答完成
4. 第 2 轮：输入 `"它是什么行业"`
5. 等待回答

### 验收标准

| 检查项 | 期望 |
|--------|------|
| 两轮 `session_id` | 一致 |
| 第 2 轮 `intent.secondary_scenario` | `industry_scan`（或主场景为 `industry_scan`） |
| 第 2 轮 `answer` | **必须包含** 000021 的行业信息（电子行业） |
| 第 2 轮 `answer` | **不应**只是重复"空仓/防守/市场不好"等泛化市场判断 |
| `remembered_stock_code` | `000021.SZ` 或 `000021` |

**❌ 不合格**：
- 第 2 轮回答与第 1 轮相同内容（没有切换视角到行业）
- 第 2 轮没有提到 000021 所属行业

### 通过标准

第 2 轮回答应类似：

> "000021（深科技）所属**电子**行业。当前电子行业……"

或至少明确提到"电子行业"或"000021 所属行业"。
