# Hermass Observer Product PRD

日期：2026-05-18
状态：V0.1_PRODUCT_SHELL

## 1. 产品定位

Hermass Observer Product 是面向普通投资者与私域用户的每日观察产品。

它回答：

```text
今天先观察什么？
为什么进入观察？
它属于哪个状态？
如果深入研究，应该看哪些证据？
```

它不回答：

```text
现在买哪只？
买多少？
何时卖？
保证收益多少？
```

## 2. 用户

第一阶段用户：

- 普通投资者
- 私域社群用户
- 需要每天快速缩小观察范围的人

第二阶段用户：

- 专业研究员
- 交易策略爱好者
- 内部回测验证人员

## 3. V0.1 功能

| 模块 | 内容 |
|---|---|
| 每日观察卡 | 展示当天观察对象、观察原因、数据边界 |
| Observer 状态 | 展示 D1/W1/MN1 三个独立观察者 |
| Omni 协调层 | 展示三个周期之间的关系与冲突类型 |
| 个股入口 | 从观察卡进入详情页，后续扩展策略、行业、资金流 |
| 发布验证 | 验证数据契约、禁用词、旧字段歧义 |

## 4. 不做什么

V0.1 不做：

- 登录系统
- 实盘委托
- 交易提醒
- 收益承诺
- 自定义策略回测
- 重新计算底层 state

## 5. 产品语言

允许：

```text
观察池
值得复核
状态仍在观察中
历史同状态路径
策略条件出现频率
```

禁止：

```text
买入
卖出
加仓
减仓
止盈
止损
荐股
收益承诺
策略已验证赚钱
```
