# Hermass Observer Brand Assets

## Files

- `hermass-observer-logo.svg`
  - 主 logo，适合 README、官网页头、文档封面
- `hermass-observer-icon.svg`
  - 图标版，适合机器人头像、favicon、社群头像

## Visual Direction

- 主色：`#14323D`
  - 稳定、理性、结构感
- 强调色：`#0E7C66`
  - 代表扩张、趋势、状态推进
- 点缀色：`#D98A1A`
  - 代表观察点、共振触发、关键提示
- 背景色：`#F5F1E8`
  - 避免金融产品常见的纯黑/纯白/廉价科技蓝

## Symbol Meaning

- 两个竖条：代表多周期状态柱
- 斜向连线：代表趋势推进与结构突破
- 外部弧线：代表观察器视角与状态边界
- 顶部圆点：代表关键观察点、提醒触发点

## Usage Notes

- 面向用户的默认版本优先用 `hermass-observer-logo.svg`
- 群机器人、钉钉头像、应用图标优先用 `hermass-observer-icon.svg`
- 不建议单独抽离文字部分使用
