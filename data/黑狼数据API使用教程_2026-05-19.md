# 黑狼数据API接口使用教程

## 简介

本教程将教您如何使用AI助手（如通义千问、文心一言、Kimi、豆包等）或小龙虾来调用黑狼数据API获取股票数据。

**官方网站**: http://www.fxyz.site/  
**开源项目**: https://github.com/fxyzsite/FxyzSkill

---

## 快速开始

### 第一步：获取Token

1. 访问 http://www.fxyz.site/apply 申请API权限
2. Token是调用API的凭证，请妥善保管，不要泄露给他人

### 第二步：选择AI助手

您可以使用以下任一AI助手：
- **通义千问** (阿里云)
- **文心一言** (百度)
- **Kimi** (月之暗面)
- **豆包** (字节跳动)
- **智谱清言** (智谱AI)
- **小龙虾** (自定义AI助手)

### 第三步：投喂API文档

将本文档内容复制并发送给AI助手，让它学习如何调用黑狼数据API。

**方式1：直接复制**
点击页面右上角的“复制全部文档”按钮，然后粘贴到AI助手中。

**方式2：下载文档**
点击下载按钮保存为markdown文件，然后将文件上传给AI助手。

## API基础信息

### 基础URL

所有API请求的基础URL为：`http://api.fxyz.site`

### 认证方式

所有API请求都需要在URL参数中包含 `token` 参数进行身份验证。

**示例：**
```
http://api.fxyz.site/wolf/list?flag=0&token=YOUR_TOKEN
```

### 返回格式

所有API返回JSON格式数据。具体返回结构请参考各接口的"返回数据示例"。

**注意：** 部分接口直接返回数组，部分接口返回对象，请以各接口的返回数据示例为准。

---

## 数据更新频率

- **实时行情数据**：交易时间段（09:15-15:00）每分钟更新
- **涨停板/跌停板/炸板数据**：实时数据交易时间段每分钟更新，历史数据收盘后3:30更新
- **基础数据（股票列表、财务指标等）**：历史数据收盘后6:00更新
- **K线数据**：实时数据交易时间段实时更新，历史数据收盘后3:30更新
- **逐笔交易/分价/资金流向**：历史数据盘后6:00更新

---

## 调用限制说明

- **免费版**：有调用次数限制，适合学习和小规模使用
- **包年版**：无调用次数限制，支持all参数获取全市场数据
- **调用频率**：建议控制调用频率，避免短时间内大量请求
- **all参数**：仅包年版用户可用，用于获取全市场数据

## 可用接口列表

### 基础数据

#### 股票列表

**接口描述**: 获取股票列表。
历史数据收盘后六点更新。

**请求URL**: `/wolf/list`

**输入参数**:

| 参数名 | 类型 | 必填 | 说明 | 取值范围 |
|--------|------|------|------|----------|
| flag | int | 否 | 股票类型 | 0-所有股票，1-深交所股票，2-上交所股票，3-北交所股票，4-指数，5-创业板股票，6-科创板股票，7-ETF，8-ST股票，9-退市股票 |
| token | String | 是 | 登录网站获取token | - |

**返回字段**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | String | 股票代码 |
| name | String | 股票名称 |

**调用示例**:

```
http://api.fxyz.site/wolf/list?flag=0&token=88888888
```

**返回数据示例**:

```json
[{"code":"000011","name":"深物业A"},{"code":"300697","name":"电工合金"}]
```

#### 涨停板

**接口描述**: 获取盘中涨停板实时数据。
数据更新：实时数据交易时间段每1分钟更新，历史数据收盘后3:30更新。

**请求URL**: `/wolf/zt`

**输入参数**:

| 参数名 | 类型 | 必填 | 说明 | 取值范围 |
|--------|------|------|------|----------|
| tradeDate | String | 否 | 交易日期 | - |
| token | String | 是 | 登录网站获取token | - |

**返回字段**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| tradeDate | String | 交易时间 |
| code | String | 股票代码 |
| name | String | 股票名称 |
| cje | float | 成交额（元） |
| scfb | String | 首次封板时间，个股第一次触及涨（跌）停的时间 |
| fbzj | float | 封板资金（元），以涨停价买入挂单的资金总量 |
| hsl | float | 换手率（%） |
| lbs | int | 连板数，个股连续封板数量 |
| zhfb | String | 最后封板时间，个股最后一次触及涨（跌）停的时间 |
| ltsz | float | 流通市值（元） |
| zxj | float | 最新价（元） |
| zsz | float | 总市值（元） |
| zbcs | int | 炸板次数，个股打开涨停板的次数 |
| zdf | float | 涨跌幅（%） |
| zttj | String | 涨停统计，n/m代表m天中有n次涨停板 |
| update | String | 更新时间 |

**调用示例**:

```
http://api.fxyz.site/wolf/zt?tradeDate=2026-04-26&token=88888888
```
