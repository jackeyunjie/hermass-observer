# 产品内核重定义

日期：2026-06-01
主题：从 EF 共振筛选转向多周期 Agent 市场态运转系统

## 关键认知转折

EF 共振筛选是整个市场态的一个组合，是参与趋势交易的一种简易方式。但整个市场态的运转才是参与的根本。

产品内核不是 EF，而是「多周期 Agent 组成的市场态运转大数据」。

## 正确的视角

```
多周期 Agent 市场态运转系统
├── MN1 Agent ──┐
├── W1 Agent  ──→ State 矩阵 ──→ 概率分布
├── D1 Agent  ──┘                    │
├── 资金流证据 →                      ├── 盈亏比预期
├── 行业承接   →                      ├── 转移方向
├── 宏观先验   →                      ├── 共振强度
│                                    └── EF 只是其中一个镜头
```

## 前端该承载什么（5 层信息）

| 层 | 内容 | 举例 |
|----|------|------|
| L1 市场态分布 | 全市场 MN1/W1/D1 的 State 构成 | 72% 在逆风期，4% 在生长季 |
| L2 转移预期 | 基于历史运转数据的概率推断 | 类似分布下，次日 58% 概率至少一半止跌 |
| L3 策略-环境匹配 | 当前市场态下各策略路径的预期质量 | 逆风主导下 VCP 假突破概率升高 |
| L4 盈亏比证据 | 不同 State 组合的 RR 分布 | EF≥2 样本的平均 RR 是 3.2:1 |
| L5 Agent 置信度 | 数据链路是否支持今天的推断 | 基础数据日更 ✓ 宏观低频但可用 ✓ |

## D1/W1/MN1 三周期 Agent 视角体系

当前系统采用 D1 Agent 为主口径：
- 所有周期的 position 计算都使用 D1 收盘价比较各自周期的 SR
- MN1 position = D1 close vs MN1 SR
- W1 position = D1 close vs W1 SR
- D1 position = D1 close vs D1 SR

未来可扩展 W1 Agent、MN1 Agent 独立视角：
- W1 Agent：用周线收盘价看周线结构
- MN1 Agent：用月线收盘价看月线结构
- 不同 Agent 的 state_hex 可以不同，这不是 bug 而是设计特性
