## 核心结论（一次性读到）

### 产品内核
Hermass 内核不是 EF 共振筛选，是**多周期 Agent 市场态运转系统**。
D1 Agent → 执行信号，W1 Agent → 趋势质量过滤，MN1 Agent → 大方向许可。

### 三周期权重
- D1：50-60%（次日回报相关性 0.205，最强）
- W1：25-30%（独立 EF 率最低 = 最严格过滤器）
- MN1：15-20%（方向最稳定）

### 五条红线（不可绕过）
1. 止损/止盈不可自动执行
2. VCP/2560/Bollinger/composite 四策略结构不可变
3. 数据异常必报人类 → AgentBus review_needed
4. 单只 25% / 单行业 40% 仓位上限
5. Admin kill-switch 24h 自动过期

### 复盘体系
`agent_self_review(4h) → agent_cross_review(每日) → alert_scanner → AgentBus.review_needed`
