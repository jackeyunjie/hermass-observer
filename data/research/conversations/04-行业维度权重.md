# 行业维度权重分析

日期：2026-06-01
主题：国内股市行业/细分行业分析的权重，以及在 State 运转体系中的位置

## 核心观点

> 整体指数对于个股的指导意义不是决定性的，行业乃至细分行业的分析才是关键。宽基指数的 State 只能告诉你'大环境能不能做'，但 A 股真正的 alpha 来自行业层面的结构差异。

## 行业在 State 运转体系中的权重

### 为什么行业比指数重要

A 股的结构性特征：
1. 行业轮动是 A 股核心驱动力——同一天内，不同行业的 State 分布差异可达 10 倍
2. 宽基指数是 5000 多只股票的加权平均，会掩盖结构差异
3. 即使上证指数下跌，电子行业可能仍在 EF 三周期共振中
4. 行业 ETF 的 State 本身就是行业层面的 MN1/W1/D1 三周期结构

### 行业在 State 运转中的五层角色

| 层 | 行业如何参与 | 权重影响 |
|----|------------|---------|
| L1 市场态分布 | 行业间 State 偏离度 = 结构机会的信号 | 行业分化越大，个股选择越重要 |
| L2 转移预期 | 行业级别 State 转移方向 vs 全市场转移方向 | 顺风行业 = 转移概率 +15-20% |
| L3 策略-环境匹配 | 同一策略在不同行业的适配度不同 | 行业是策略适配的第二道过滤 |
| L4 盈亏比证据 | 行业内 EF 样本的平均 RR vs 跨行业 RR | 行业内 RR 方差 < 跨行业 RR 方差 |
| L5 Agent 置信度 | 行业 ETF 的 MN1 score 决定仓位系数 | 行业 MN1 直接影响 position sizing |

### 行业在 SW 三级体系中的角色

| 层级 | 数量 | 作用 | 在系统中的当前使用 |
|------|------|------|-----------------|
| SW 一级行业 | 31 个 | 大类方向（电子、医药、消费...） | industry_rotation 使用 |
| SW 二级行业 | ~130 个 | 细分板块（半导体、消费电子...） | unified snapshot 存储但未在页面上显性使用 |
| SW 三级行业 | ~350 个 | 精细赛道（被动元件、面板...） | 仅在 research 页面展示，未用于筛选 |

**当前缺失**：二级和三级行业尚未作为独立的筛选维度参与 State 运转。

## 行业与三周期 Agent 的最终权重框架

将行业维度的分析纳入此前三周期权重框架：

| 维度 | 权重 | 作用 | 数据支撑 |
|------|------|------|---------|
| **D1 Agent**（执行） | **40-50%** | 今天能不能动 | 与次日回报 r=0.205 |
| **W1 Agent**（过滤） | **20-25%** | 行情有没有骨架 | 最严格瓶颈，行业间差异最显著 |
| **MN1 Agent**（方向） | **10-15%** | 方向上仓位多大 | 波动最小，行业 MN1 差异稳定 |
| **行业维度**（新增） | **15-25%** | 方向选对没有 | 行业内 EF2 密度 vs 全市场均值 |
| **细分行业** | 行业内的加权 | 同行业挑对赛道 | 半导体 85% vs 元件 73% 等二级差异 |

### 行业修正规则

1. **行业内 EF2 密度 > 全市场 2 倍**（>21.4%）→ 该行业信号加权 +1
2. **行业 ETF W1/MN1 State = E/F** → 行业内所有信号适配度+1
3. **行业 ETF D1 State = 逆位** → 行业内信号自动降级为观察
4. **行业 MN1 系数** → 直接影响该行业内个股的仓位系数

## 具体建议

### 1. 前端层面：行业作为第一导航维度
- 首页不应该从'市场能不能做'开始，而应该从'哪些行业有结构机会'开始
- 市场宽度是背景，行业热度是行动入口

### 2. 信号层面：行业为信号加权
- EF=3 + 行业 ETF MN1=E → 信号置信度 +1
- EF=2 + 行业 ETF W1=逆位 → 降级为观察
- 行业内 EF 密度 > 全市场 2 倍 → 该行业信号优先

### 3. 数据层面：补齐细分行业
- SW 二级/三级行业应作为独立的筛选维度
- 行业内个股之间的 State 差异比跨行业更有参考意义

### 4. 后验层面：行业级盈亏比
- 不同行业的 EF3 → 次日收益分布不同
- 顺风行业 EF3 的胜率 > 逆风行业 EF3
- 这比全市场平均 RR 更有指导意义
