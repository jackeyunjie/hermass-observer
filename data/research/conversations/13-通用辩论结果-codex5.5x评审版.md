## 第一轮：找分歧点（逐题）

### Q1 产品内核
- Codex：内核 = 多周期 Agent 市场态运转系统。
- Claude：内核 = 多周期 Agent 市场态运转系统；EF 是输出。
- Kimi：内核 = EF 共振筛选，运转系统是包装。

### Q2 三周期权重
- Codex：W1 0.45, D1 0.35, MN1 0.20（静态基线，规则切换）。
- Claude：D1/W1/MN1 = 35/35/30（动态，Market Entropy 驱动）。
- Kimi：D1 0.45, W1 0.30, MN1 0.25（相关性+信号独特性）。

### Q3 行业权重
- Codex：二级门控，固定 15%，二级行业仅异常修正。
- Claude：上下文门控/贝叶斯乘子；二级行业必须独立用于信号修正。
- Kimi：概率乘子 α=0.3；一级宏观感知+二级修正。

### Q4 升级路径
- Codex：4 阶段，JSON Schema + 500ms 截止 + 版本化 prompt。
- Claude：4 阶段，强调“行为可预测”先于“正确”，前向账本升级。
- Kimi：4 阶段，强调 15:00 批量事件和依赖环检测。

### Q5a 兵法与工程
- Codex：知胜有五映射环境感知层；“君不御”= 运行时规则固化+日级配置。
- Claude：三层权限模型（人→仲裁→Agent）；5 条红线+5 个自主领域。
- Kimi：三层权限模型类似 Claude，额外强调“每周强制人工覆盖”。

### Q5b 道德经映射
- Codex：大=MN1, 逝=W1, 远=D1, 反=D1 中识别 MN1 不认可的短期背离。
- Claude：大→逝→远→反=三周期层级；“反”是 EF2 三周期收敛。
- Kimi：量化“远曰反”为三周期方差骤降+State converged 加权。

### Q6 因子体系
- Codex：时间/事件/结构三必须；约束搜索树+条件 MI；每日最多调 3 个因子。
- Claude：三层因子（结构固定+时间半固定+自发现流动）；Shannon 熵扫描驱动发现。
- Kimi：时间/事件/结构；Pearson+交互项搜索；边际贡献±5%/季度重置。

### Q7 收缩观测
- Codex：收缩三重交叉；突破三选二+不续接延迟 3 日。
- Claude：收缩三重（BB+SR+ATR 相对分布）；突破四重（V1 基础+V2/V3/V4 加权）。
- Kimi：收缩三重；突破四重与 Claude 几乎一致。

### Q8 构建验证与运行
- Codex：4 层+混沌；fail_count 死亡计数器；JSON Schema；500ms 超时。
- Claude：5 层金字塔；人类拔线权+Arbiter。
- Kimi：4 层+混沌；Redis Streams；人类+自评+互评节奏。

### Q9 量变到质变
- Codex：胜率+回撤+行业 EF2>70%+三周期共振；贝叶斯变点+Simpson+监护期。
- Claude：Hosmer-Lemeshow+Sharpe+涌现指标；Chow 检验。
- Kimi：与 Claude 相近（Chow 检验），强调 500+ 样本+校准度无偏。

### Q10 外部方案
- Hermes-agent：Codex 引入自学习+Skill；Claude 改造深度大暂不优先；Kimi 直接引入 Skill 自动创建。
- agentmemory：Codex 需改造为时序状态检索；Claude 满分引入；Kimi 改造为 State Graph Retrieval。
- Harness：Codex/Claude 支持配置层引入；Kimi 强烈反对（金融需精确控制）。
- Karpathy：Codex 改造为 Web UI+API；Claude 直接映射 L1-L2-L3；Kimi 直接引入。

### 专属问题
- Codex-2：仅 Codex 给出完整计算架构（预计算+物化视图+增量窗口）。
- Codex-3：仅 Codex 给出死亡计数器+四层降级+恢复伪代码。
- Claude/Kimi 在专属问题上部分覆盖于 Q7/Q8。

## 第四轮：加分题（三个可能被遗漏的视角）

### 视角 1：数据质量主权层
三个模型都假设数据干净。需要前置一层 `data_quality_score`：
- `is_suspended`
- `is_limit_up / is_limit_down`
- `is_st`
- `is_ipo_first_day`

分数低于阈值时，该股票当日所有指标标记 `DEGRADED`，只记录不参与决策。

### 视角 2：因子权重免疫稳态
三个模型都追求“最优拟合”修正。建议引入生物学稳态：
- 每个因子有“耐受带”（±10% 波动不触发调整）。
- 连续 N 日超耐受带才触发“免疫反应”。
- 保留“记忆细胞”因子库，特定市场范型下快速唤醒。

### 视角 3：展示层认知卸载
Q1 讨论首页展示什么，但都停留在“信息密度优化”。更本质的问题：
- 首页不应是仪表盘，而应是**决策分级器**。
- L1（秒级）：今日 State 从非 EF 变 EF 的数量变化。
- L2（分钟级）：EF2 股票中触发策略数。
- L3（日级）：需要人工介入的异常案例数。
- 首页只显示 L1+L2+L3 的红黄绿灯，不展示 5510 只股票明细。
